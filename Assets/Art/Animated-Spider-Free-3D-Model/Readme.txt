Controllable spider for the Blender Game Engine. (Blender2.62)

Controls:
W,A,S,D = walk
A,D = side step
walk + Space = run
Left Mouse Button = attack 
Middle Mouse Button = jump
T = Die-animation 1
R = Die-animation 2
1,2,3 =camera change
